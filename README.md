# Grade Huh Moe

面向北京大学医学部同学的成绩查询与课程信息展示应用。

## ✨ 功能

-   支持多种数据来源：
    -   医学部
    -   本部
    -   混合模式（医学部提供医学部成绩，本部提供本部成绩，后者覆盖前者，更及时）
-   双学位友好，支持同一学期内成绩混合 / 分列展示

## 🧠 成绩获取方式

-   本部成绩在前端通过 IAAA 接口获取。
-   医学部成绩通过请求我开源并部署的 [Python 后端服务器](https://github.com/zhuozhiyongde/Grade-Huh-Moe-Backend) 获取，以解决跨域问题
-   您也可以 `clone` 本仓库或后端仓库，自行部署前后端
-   我承诺不会存储、发送任何您的登录凭证或敏感信息到其他非官方服务器

## 🚀 本地开发

-   `bun install` 安装依赖。
-   `bun run dev` 启动开发服务器。
